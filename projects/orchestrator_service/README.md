# orchestrator_service

Сервис-оркестратор на NestJS для управления состоянием и данными с использованием конечного автомата

## Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

## Конфигурация

Заполни `.env` на основе `.env.example`.

## Запуск

```bash
python main.py
```

## Зависимости

- python-dotenv
- httpx

## Создан

Проект создан автоматически агентом `ProjectCreatorAgent` системы `my_personal_ai`.
