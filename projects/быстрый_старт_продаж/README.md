# быстрый_старт_продаж

Пакет из 10 объявлений и визуалов для быстрого старта продаж

## Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

## Конфигурация

Заполни `.env` на основе `.env.example`.

## Запуск

```bash
python main.py
```

## Зависимости

- python-dotenv
- httpx
- python-telegram-bot

## Создан

Проект создан автоматически агентом `ProjectCreatorAgent` системы `my_personal_ai`.
