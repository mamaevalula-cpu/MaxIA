# saas_api_sales_materials

Create API documentation, pricing tiers, onboarding flow and Direct/SaaS sales materials

## Установка

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

## Конфигурация

Заполни `.env` на основе `.env.example`.

## Запуск

```bash
python main.py
```

## Зависимости

- python-dotenv
- httpx
- fastapi
- uvicorn

## Создан

Проект создан автоматически агентом `ProjectCreatorAgent` системы `my_personal_ai`.
